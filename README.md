# 🏆 Risiko oppi Spel

A highly customizable, fully responsive **Jeopardy-style trivia game** designed for immersive TV play modes and easy game management.

![License](https://img.shields.io/badge/license-MIT-purple)

## 🌟 Overview

**Risiko oppi Spel** is a production-grade trivia platform that brings the excitement of quiz shows to any screen. Whether you're hosting a local game night, a classroom activity, or a corporate event, this app provides everything you need to create, manage, and play engaging trivia games.

### ✨ Key Features

- **📺 Immersive TV Mode**: A beautiful, full-screen game board designed for large displays.
- **📱 Real-time Buzzer System**: Players join via QR code and use their smartphones as buzzers with instant synchronization.
- **🪄 AI Game Generator**: Instantly generate complete trivia boards on any topic using advanced AI.
- **🛠️ Powerfull Editor**: Customize every aspect of your game—categories, questions, points, and themes.
- **🎨 Dynamic Themes**: Choose from several professional presets or create your own custom visual style.
- **🎙️ Custom Buzzer Sounds**: Players can record or upload their own unique buzzer sounds.
- **⏲️ Smart Timer**: Configurable timers for questions and buzzers to keep the game moving.

---

## 🚀 How to Play

### 1. Create Your Game
Use the **Editor** to build your trivia board. You can manually enter categories and questions or use the **AI Generator** to bootstrap a game in seconds.

### 2. Enter the Lobby
Once your game is ready, switch to the **Play Mode**. Show the **QR Code** on the main screen for players to scan.

### 3. Players Join
Contestants scan the QR code with their mobile devices to enter their nicknames and (optionally) record a custom buzzer sound.

### 4. Start the Show!
The host selects questions from the board. When the question appears, players buzz in to answer. The host awards or deducts points based on the response.

## 🛠️ Advanced Settings

- **Buzzer Delay**: Prevent "snap buzzes" by setting a delay after the question appears before buzzers are enabled.
- **Timer Options**: Enable/disable countdown timers and choose if they start immediately or only after a buzz.
- **Bonus Questions**: Create "Daily Double" style bonus questions that can be triggered randomly or on specific rows.
